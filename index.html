Teleporter

This is my Final Year Project and it takes 4 months to be done by individually. It is a 3D action-adventure game which developed by Unity3D and the programming language used is C#. Player needs to control the game character to move in the maze and use the magical orb to teleport in the game. The gameplay is that player needs to avoid the traps in the game and find the key or solve the puzzle to unlock the portal to escape from the maze to next level. Each level has increase the challenge by adding new traps, more keys require to unlock the portal and more types of puzzles.

Level 1 with less traps

Player has to shoot the magical orb to teleport to its position

Teleport effects

Key that is put somewhere in the maze

Treasure box that contain puzzle game

Portal which will appear when puzzle is solved or key is collected

Malaysian Bro

It is a 2D platformer game which developed by Unity3D and the programming language used is C#. The game has 4 levels. The player has lives, health and able to use weapon in the game. Each level has increase the challenge by adding new enemies and new layout of the game world. For the first 3 levels, there is a small boss that player needs to defeat it to unlock the door which can go to the next level. The objective of the game is to pass every level until level 4 and beat the boss to win the game.

Level 1 with Chinese Theme

Level 1 Boss with low health point and attack rate

Level 2 with Malay Theme

Level 2 Boss with medium health point and medium attack rate

Level 3 with Indian Theme

Level 3 Boss with high health point and high attack rate

Level 4 Boss with high health point, high attack rate, it is moving around the level and has ability to jump. After first dead the boss will split into 2 and if player defeat it player will win the game

Clash Nations

It is a 2D strategy game which developed by Unity3D and the programming language used is C#. The game requires player to use the mouse cursor to pick up coin and deploy troops to defend and attack the enemy tower.

Example of in game screen

Blackout

It is a simple casual game developed by using Processing 3 and the programming language used is C++. The gameplay is player needs to control the pokeball to collect the windows icon items which will randomly spawn on the platform in the game. Black screen will continuously fall down in the game and when each time the player collects the items then the black screen will move upward. When the the game is full of black screen then the player will lose the game.
